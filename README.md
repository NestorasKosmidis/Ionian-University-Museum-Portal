# Ionian University – Museum Portal (Custom WordPress Theme)

Το παρόν αποθετήριο περιλαμβάνει τον κώδικα που αναπτύχθηκε στο πλαίσιο πτυχιακής εργασίας του Τμήματος Πληροφορικής του Ιονίου Πανεπιστημίου.

Η εργασία αφορά τον σχεδιασμό και την υλοποίηση διαδικτυακής εφαρμογής για την ψηφιακή προβολή και διαχείριση μουσειακών εκθεμάτων, μέσω προσαρμοσμένου θέματος (custom theme) σε περιβάλλον WordPress.

## Περιεχόμενο αποθετηρίου

Το αποθετήριο περιλαμβάνει:

- Τον κώδικα του προσαρμοσμένου WordPress theme
- Τα αρχεία προτύπων (templates) για:
  - σελίδα συλλογής εκθεμάτων
  - σελίδα μεμονωμένου εκθέματος
- Ρυθμίσεις και λειτουργικότητες που υποστηρίζουν:
  - δομημένη καταχώρηση εκθεμάτων
  - φιλτράρισμα και πλοήγηση περιεχομένου
  - προβολή πολυμεσικού και τρισδιάστατου υλικού

Ο κώδικας παρέχεται **ενδεικτικά και για ακαδημαϊκή χρήση**.

## Βάση δεδομένων και περιεχόμενο

Στο πλαίσιο της πτυχιακής εργασίας χρησιμοποιήθηκαν συγκεκριμένες δομές βάσης δεδομένων (custom post types, προσαρμοσμένα πεδία κ.λπ.) για την καταχώρηση των μουσειακών εκθεμάτων.

Σημαντική διευκρίνιση:
- Το περιεχόμενο που εμφανίζεται στην εφαρμογή κατά την ανάπτυξη της εργασίας αφορούσε δοκιμαστικά δεδομένα.
- Σε νέα εγκατάσταση WordPress, το περιεχόμενο θα πρέπει να καταχωρηθεί εκ νέου από τον διαχειριστή ή να χρησιμοποιηθούν πρόσθετα όπως "duplicator" για πλήρη μεταφορά

Η παρεχόμενη δομή επιτρέπει σε έναν πολιτιστικό οργανισμό ή μουσείο να εγκαταστήσει το theme και να δημιουργήσει το δικό του περιεχόμενο, αξιοποιώντας τη λειτουργικότητα που έχει υλοποιηθεί.

## Εγκατάσταση

Το προσαρμοσμένο theme μπορεί να εγκατασταθεί σε νέα εγκατάσταση WordPress ως εξής:

1. Κατεβάστε το αποθετήριο σε μορφή ZIP
2. Από το WordPress Administration Panel:
   - Appearance → Themes → Add New → Upload Theme
3. Μεταφορτώστε το αρχείο ZIP και ενεργοποιήστε το theme
4. Προσθέστε τα εκθέματα και το περιεχόμενο μέσω του διαχειριστικού περιβάλλοντος

## Σκοπός 

Ο κώδικας αναπτύχθηκε **αποκλειστικά για εκπαιδευτικούς και ακαδημαϊκούς σκοπούς**, στο πλαίσιο πανεπιστημιακής εργασίας.

Μπορεί να χρησιμοποιηθεί:
- για μελέτη
- για εκπαιδευτικά παραδείγματα
- ως βάση για περαιτέρω ανάπτυξη

Δεν προορίζεται ως έτοιμο εμπορικό προϊόν.
